Semester Project 2

Mercury Shoes

Mercury shoes is an e-commerce site selling running shoes. The site is coded using HTML and vanilla Javascript, and is styled using Sass. The site has a costumer facing side with basket functionality using local storage to store data, and also an admin facing side, where products can be added, edited and deleted. An API from Strapi hosted on Heroku is used to store and fetch content data.

Project Exam 2

Holidaze

Holidaze is an accomodation booking site for a local tourist agancy located in Bergen. The site is coded with ReactJS based on React Bootstrap components and styled using Sass. The site has a customer facing side with booking functionality, and a admin section that can be accessed with the use of JWT tokens. In the admin section, contact and booking enquiries can be handled as well as adding new establishments. The WP WooCommerce API is used for the establishments and customer reviews, while a API from Strapi hosted on Heroku is used to store and fetch data from booking- and contact enquiries.

JS Frameworks CA

Amazing Products

This JavaScript frameworks course assignment is coded with ReactJS and styled using Sass. It has login functionality that make use of JWT tokens from the Wordpress API. For the content it fetches some random products from the Fake Store API, and displays them in a grid. For each product there is also a details page.
